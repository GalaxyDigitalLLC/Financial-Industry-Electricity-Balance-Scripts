Accompanying calculations scripts for *On Bitcoinâ€™s Energy Consumption: A Quantitative Approach to a Subjective Question* by the Galaxy Digital Mining Team.


## Dependencies
The following dependencies are required to compile these scripts:
- python3
- pytest


## Build
  Install dependencies:
  ```
  $ make install
  ```

  Run the python scripts:
  ```
  $ make
  ```

  Test the python scripts:
  ```
  $ make test
  ```

  Run the python scripts in watch mode for development purposes:
  ```
  $ make watch
  ```

  Run the python scripts tests in watch mode for development purposes:
  ```
  $ make watch-test
  ```

  Lint:
  ```
  $ make lint
  ```
